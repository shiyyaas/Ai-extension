# Lumina - Smart Reading Assistant

**Lumina** is a smart reading assistant that helps you understand text on any webpage. Powered by on-device AI, Lumina ensures your data remains private while providing powerful tools to simplify your reading experience. Whether you're a student, a professional, or just a curious reader, Lumina is designed to make information more accessible.

## Key Features

- **On-Device AI:** All text processing is done locally on your device, ensuring your data remains private and secure.
- **Instant Summaries:** Quickly get the gist of any article or document with a concise summary.
- **Clear Explanations:** Understand complex subjects with simple, easy-to-understand explanations.
- **Simplified Language:** Rewrite complicated text into a more readable and accessible format.
- **Floating Action Button:** A convenient button appears whenever you select text, giving you quick access to all of Lumina's features.

## Getting Started

### Prerequisites

- A modern web browser that supports Web AI (e.g., Chrome, Edge).

### Installation and Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/shiyaz-ahmed/Lumina.git
   ```

2. **Load the Extension:**
   - Open your browser's extension management page (e.g., `chrome://extensions`).
   - Enable **Developer mode**.
   - Click **Load unpacked** and select the cloned `lumina` directory.

3. **Download the AI Model:**
   - Once the extension is loaded, click on the Lumina icon in your browser's toolbar.
   - If the on-device AI model is not available, you will be prompted to download it. Click the **Download AI Model** button and wait for the download to complete.

### How to Use

1. **Select Text:** Highlight any text on a webpage that you want to understand better.
2. **Activate Lumina:** A floating Lumina button will appear next to your selected text. Click on it to open the action menu.
3. **Choose an Action:** Select one of the available actions:
   - **Summarize:** Get a concise summary of the text.
   - **Explain:** Receive a simple explanation of the topic.
   - **Simplify:** Rewrite the text in easier-to-understand language.
4. **View the Result:** The result will be displayed in a clean, easy-to-read popup.
